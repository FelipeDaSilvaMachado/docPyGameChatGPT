Documentação Explicativa e Exemplos de Funções do Pygame

Inicialização e Configuração Básica

  • pygame.init()
    Inicializa todos os módulos do Pygame (gráficos, sons, etc.), deixando-os prontos para uso. 
    import pygame\npygame.init()
  • pygame.quit()
    Encerra todos os módulos do Pygame. 
    pygame.quit()

Tela e Desenho

  • pygame.display.set_mode((largura, altura))
    Cria uma janela de jogo. 
    screen = pygame.display.set_mode((720, 480))
  • pygame.display.set_caption("Título")
    Define o título da janela. 
    pygame.display.set_caption("Meu Jogo")
  • screen.fill(cor)
    Preenche a tela com uma cor específica. 
    screen.fill("purple")
  • pygame.display.flip()
    Atualiza toda a tela. 
    pygame.display.flip()
  • pygame.display.update([rects])
    Atualiza áreas específicas da tela. 
    pygame.display.update()

Controle de Tempo

  • pygame.time.Clock()
    Cria um objeto Clock para controlar a taxa de quadros. 
    clock = pygame.time.Clock()
  • clock.tick(FPS)
    Define a taxa de quadros por segundo. 
    clock.tick(60)

Eventos e Entrada

  • pygame.event.get()
    Retorna uma lista de eventos. 
    for event in pygame.event.get():\n    if event.type == pygame.QUIT:\n        running = False
  • pygame.key.get_pressed()
    Verifica teclas pressionadas. 
    keys = pygame.key.get_pressed()\nif keys[pygame.K_LEFT]:\n    print("Esquerda!")
  • pygame.mouse.get_pos()
    Retorna a posição do mouse. 
    mouse_pos = pygame.mouse.get_pos()

Sons e Música

  • pygame.mixer.init()
    Inicializa o mixer de áudio. 
    pygame.mixer.init()
  • pygame.mixer.Sound('arquivo.wav')
    Carrega um efeito sonoro. 
    som = pygame.mixer.Sound('beep.wav')
  • sound.play()
    Reproduz o som carregado. 
    som.play()
  • pygame.mixer.music.load('musica.mp3')
    Carrega uma música de fundo. 
    pygame.mixer.music.load('musica.mp3')
  • pygame.mixer.music.play(loops=0)
    Reproduz a música. 
    pygame.mixer.music.play(-1)

Desenho de Formas

  • pygame.draw.rect(surface, cor, (x, y, largura, altura))
    Desenha um retângulo. 
    pygame.draw.rect(screen, "blue", (50, 50, 100, 50))
  • pygame.draw.circle(surface, cor, (x, y), raio)
    Desenha um círculo. 
    pygame.draw.circle(screen, "red", (200, 200), 30)
  • pygame.draw.line(surface, cor, ponto_inicial, ponto_final, espessura)
    Desenha uma linha. 
    pygame.draw.line(screen, "green", (300, 300), (400, 400), 5)

Textos

  • pygame.font.Font(nome_da_fonte, tamanho)
    Carrega uma fonte. 
    font = pygame.font.Font(None, 36)
  • font.render(texto, anti_aliasing, cor)
    Renderiza texto. 
    text_surface = font.render("Olá, Pygame!", True, "black")\nscreen.blit(text_surface, (150, 150))

Essas são as funções e chamadas mais comuns no Pygame, acompanhadas de exemplos de uso para facilitar o entendimento.
