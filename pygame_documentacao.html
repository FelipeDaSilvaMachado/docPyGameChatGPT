Documentação Explicativa de Funções do Pygame

Inicialização e Configuração Básica

  • pygame.init()
    Inicializa todos os módulos do Pygame (gráficos, sons, etc.), deixando-os prontos para uso.
  • pygame.quit()
    Encerra todos os módulos do Pygame. Deve ser chamada no final do jogo para liberar os recursos.

Tela e Desenho

  • pygame.display.set_mode((largura, altura))
    Cria uma janela de jogo com as dimensões especificadas em pixels (ex.: (720, 480)). Retorna um objeto Surface que representa a tela do jogo, permitindo desenhar elementos nela.
  • pygame.display.set_caption("Título")
    Define o título da janela do jogo.
  • screen.fill(cor)
    Preenche toda a tela com uma cor específica. A cor pode ser especificada como uma string ("purple") ou como uma tupla RGB ((128, 0, 128)).
  • pygame.display.flip()
    Atualiza toda a tela para refletir as mudanças feitas. Normalmente chamado após desenhar elementos para exibir o frame completo.
  • pygame.display.update([rects])
    Atualiza apenas as áreas especificadas (retângulos) da tela. Isso pode ser mais eficiente que flip() para pequenos ajustes.

Controle de Tempo

  • pygame.time.Clock()
    Cria um objeto Clock que permite controlar a taxa de quadros do jogo.
  • clock.tick(FPS)
    Limita o loop do jogo para rodar a uma taxa de frames por segundo (ex.: clock.tick(60) limita para 60 FPS).

Eventos e Entrada

  • pygame.event.get()
    Retorna uma lista de todos os eventos na fila de eventos do Pygame. Eventos incluem fechamento da janela, teclas pressionadas, etc.
  • pygame.event.clear()
    Limpa todos os eventos da fila de eventos, útil para reiniciar o estado de entrada.
  • pygame.key.get_pressed()
    Retorna uma lista com o estado de todas as teclas do teclado (True se pressionada, False se não). Útil para verificar teclas mantidas pressionadas.
  • pygame.mouse.get_pos()
    Retorna a posição atual do cursor do mouse como uma tupla (x, y).
  • pygame.mouse.get_pressed()
    Retorna uma lista indicando se os botões do mouse estão pressionados (True) ou não (False).

Sons e Música

  • pygame.mixer.init()
    Inicializa o mixer de áudio para reproduzir sons.
  • pygame.mixer.Sound('arquivo.wav')
    Carrega um efeito sonoro de um arquivo. Retorna um objeto Sound que pode ser reproduzido.
  • sound.play()
    Reproduz o som carregado (usado com um objeto Sound).
  • pygame.mixer.music.load('musica.mp3')
    Carrega uma música de fundo de um arquivo.
  • pygame.mixer.music.play(loops=0)
    Reproduz a música carregada. O parâmetro loops indica quantas vezes a música será repetida (-1 para repetir indefinidamente).

Desenho de Formas

  • pygame.draw.rect(surface, cor, (x, y, largura, altura))
    Desenha um retângulo de uma cor específica na superfície indicada (ex.: screen). (x, y, largura, altura) define a posição e o tamanho do retângulo.
  • pygame.draw.circle(surface, cor, (x, y), raio)
    Desenha um círculo de uma cor específica na superfície indicada. (x, y) define o centro do círculo e raio o tamanho.
  • pygame.draw.line(surface, cor, ponto_inicial, ponto_final, espessura)
    Desenha uma linha de uma cor específica entre dois pontos, com uma espessura específica.
  • pygame.draw.polygon(surface, cor, pontos)
    Desenha um polígono de uma cor específica na superfície indicada. pontos é uma lista de tuplas, cada uma representando um vértice do polígono.

Textos

  • pygame.font.Font(nome_da_fonte, tamanho)
    Carrega uma fonte com o tamanho especificado. nome_da_fonte pode ser None para a fonte padrão.
  • font.render(texto, anti_aliasing, cor)
    Renderiza um texto usando a fonte carregada. anti_aliasing define se o texto será suavizado (True ou False). Retorna uma superfície com o texto renderizado.
